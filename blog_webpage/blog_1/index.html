Resources for learning how to code

Author: Sam Moore

One of the most valuable skills you can have today in any scientific field is the ability of program. This is especially true in Astrophysics where one can be analysing large datasets (processed from telescopes) or conducting computational simulations (e.g: modelling plasmas). Because of this, I have collected a number of resources that are useful to the beginner who wishes to gain coding experience. Be warned though, simply browsing the odd website every few months in moments of motivation is not enough! Learning requires doing, especially in programming, so be sure to use these skills regularly and solve coding problems.

SQL

SQL is a common language needed to pull information from databases. It’s quite simple, and to that effect I think Codecademy does a good job.

https://www.codecademy.com/learn/learn-sql - especially the queries section

C

C is an older programming language that is still highly popular in the world today, and is often seen in certain physics projects. I commonly see this when looking at large computational simulations (in plasma physics and cosmological models).

To learn C, I would recommend this website: https://www.learn-c.org/en/Welcome

C doesn’t require an IDE like Python, but there is apparently a way of coding in C with Jupyter notebook if you like:

https://xuedong.github.io/programming/2017/12/27/jupyter-c-kernel.html

MATLAB

MATLAB is a relatively popular programming language in Physics, and is usually picked because of how the language treats vectors and matrix operations. You have to pay to download MATLAB, and it’s hideously expensive, so unless you’re a student of a university where you can download their version for free, don’t bother. In my experience MATLAB is also a marmite language: you either love it or you hate it. I am in the dislike camp as I find Python to be a lot more intuitive, but in case you want/need to learn, have a look at: https://matlabacademy.mathworks.com/

Java

Java is another popular language. I personally haven’t seen it used in scientific research at all, but I have had to learn it for computer science classes.

Then, there are a couple of ways to learn Java. I personally used the textbook Computer Science: An Interdisciplinary Approach by Robert Sedgewick and Kevin Wayne, as this also teaches concepts in computer science.
